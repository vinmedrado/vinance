# Financial Memory Engine

O Vinance mantém uma memória financeira por `organization_id` e `user_id` usando snapshots mensais já isolados por tenant.

A memória identifica padrões mensais, meses críticos, sazonalidade, categorias sensíveis, evolução de reserva, consistência de aportes e tendência financeira.

Saída principal:
- `memory_strength`
- `patterns`
- `critical_months`
- `critical_categories`
- `trend`
- `stability_index`
- `insights`

Disclaimer: o Vinance fornece análises educacionais baseadas em dados históricos e modelos quantitativos. Isso não constitui recomendação financeira.
