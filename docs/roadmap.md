# Roadmap do FinanceOS

## MVP atual

- Dashboard Streamlit com visão geral financeira
- Catálogo e cobertura de ativos
- Automação de dados de mercado
- Scores, rankings e ranking final calibrado
- Backtests, otimizações e comparação de estratégias
- Orquestrador geral de rotinas
- Jobs e execuções operacionais
- Módulos iniciais de ML e inteligência
- Autenticação, planos, API keys e healthcheck

## Próximos módulos

- Testes automatizados por domínio crítico
- Cobertura maior para ações, FIIs, ETFs, BDRs, cripto e ativos globais
- Validação de qualidade de dados com relatórios mais visuais
- Mais estratégias de backtest com controle de risco
- Monitoramento operacional com alertas
- Documentação de APIs e fluxos de uso

## Melhorias futuras

- Feature store para modelos de ML
- Rebalanceamento com simulação de custos e impostos
- Gestão avançada de carteira
- Cenários macroeconômicos
- Deploy com ambiente de homologação
- Observabilidade com logs estruturados, métricas e tracing

## Possíveis integrações

- Provedores de dados financeiros pagos ou gratuitos
- APIs de corretoras, quando permitido
- WhatsApp, e-mail ou Telegram para alertas
- BI externo como Power BI
- Armazenamento em nuvem
- Pipelines agendados em GitHub Actions ou workers dedicados
