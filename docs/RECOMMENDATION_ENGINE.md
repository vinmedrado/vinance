# Motor de Recomendação

Serviço principal: `InvestmentRecommendationService`.

Ele combina:

- perfil financeiro;
- capacidade mensal de aporte;
- alocação inteligente;
- scoring contextual de ativos;
- backtest personalizado;
- metas financeiras.

Resposta esperada:

- recomendação simples;
- aporte mensal sugerido;
- alocação percentual;
- risco;
- estimativa de meta;
- comparação com benchmarks;
- cenários;
- explicações humanas;
- disclaimer financeiro.

Nenhuma recomendação deve ser apresentada como garantia de retorno.
