# Advanced Forecast

O forecast projeta patrimônio, reserva, capacidade futura, risco financeiro e cenários.

Cenários avançados:
- conservador
- moderado
- agressivo
- crise
- inflação alta
- juros altos

Por compatibilidade, chamadas simples continuam retornando pessimista/base/otimista. O modo avançado é ativado com `include_advanced=True`.
